# AI Video Service Web App

Это веб-приложение на **Python (Flask) + HTML/CSS/JS**, которое позволяет пользователям заказывать реалистичные видео, фото и услуги с ИИ, а также общаться в чате. Приложение поддерживает **русский и узбекский интерфейсы**, а также защищённую **админ-панель**.  

---

## Функционал

### Пользователь:
- Регистрация и вход
- Выбор языка интерфейса (RU/UZ)
- Создание заказов на услуги
- Отправка сообщений и файлов в чате
- Просмотр статуса заказов

### Админ:
- Вход только по логину `Bobur2012.12` и паролю `4348888b`
- Управление заказами (одобрение, отклонение, удаление)
- Управление услугами (добавление, удаление)
- Просмотр всех сообщений и заказов
- Полная защита админ-панели (пользователи не могут попасть через URL)
